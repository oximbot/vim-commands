# Vim commands

A collection of Vim commands that every developer should know.

---

## Modes
| commands | Description |
| -------- | ----------- |
| i | insert mode |
| a | insert mode |
| esc | back to normal mode |
| v / V | visual mode |

---

## Movement
|---------------|-------------------------|
| h / j / k / l | move left down up right |
| w | next word |
| b | previous word |
| ^ | start of the line |
| $ | end of the line |
| G | go to the line (10G, 1G, gg(top of file)) |

---

## Editing
|---|-----------------|
| d | delete (dw, dd) |
| y | yank/copy (yy, yw, ..) |
| p | paste |
| u | undo |
| crtl + r | redo |
| x / X | delete one charecter |
| o / O | open new line |
| r | replcae |
| j | join |
| a | append |
| . | repeat |

---

## Search
|----|--------------|
| /X | search for X |
| N / n | next and previous result |

---

## Save & Quit
|----|------|
| :w | Save |
| :q | quit |
| :wq | Save and quit |
| :q! | force quit |

---

## Why Vim?

Learning Vim improves:
- Editing speed
- Terminal workflow
- Keyboard efficiency
- Focus
